## HelpNaut

Astronaut mental health assistant 

## Stack

- **Backend:** Flask, Google Gemini API
- **Frontend:** React, Vite

## Setup

## Backend

python -m venv venv
source venv/bin/activate 
pip install -r requirements.txt

add API key to .env file

GOOGLE_API_KEY=your_key_here

## Frontend

cd web
npm install
